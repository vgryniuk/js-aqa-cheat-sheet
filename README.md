# js-aqa-cheat-sheet
Короткий конспект по javaScript для AQA

---
**JavaScript** - млва програмування створена компанією Netscape для використання у браузерах та надання можливості динамічно маніпулювати вмістом сторінок, що було актуально для розвитку веб-технологій у 1990-2000 роках.

Основними особливостями JS є:
* Динамічна типізація
* Асинхронний код і колбеки
* Об'єктно-орієнтований підхід
* DOM (Document Object Model) маніпуляції
* Легкість використання
